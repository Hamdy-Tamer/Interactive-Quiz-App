# Interactive Quiz Application

A modern, responsive quiz application built with vanilla JavaScript that supports both single-answer and multiple-answer questions. Perfect for testing knowledge with immediate feedback and detailed review mode.


## ‚ú® Features

- **Dual Question Types**: Support for both single-answer (radio buttons) and multiple-answer (checkboxes) questions
- **60-Second Timer**: Countdown timer with visual warning when time is running low
- **Progress Tracking**: Bullet indicators showing answered, current, and pending questions
- **Review Mode**: After completion, review all answers with correct/incorrect highlighting
- **Clear Choices Button**: Reset your selections for the current question with one click
- **Score Display**: Final score shown in an animated percentage circle
- **Fully Responsive**: Works seamlessly on mobile, tablet, and desktop devices
- **Touch-Friendly**: Optimized for touch interactions on mobile devices
